# Magic & Spell Lores


### Lore Structure

Each spell lore contains:

- **Signature spell:** known for free by every wizard with access to the lore. Does not count against Spells Known.
- **Lore attribute:** a passive effect that applies whenever casting from this lore. Gives each lore mechanical identity.
- **Basic (Tier 1):** 3-4 spells, requires Lore Access 1+.
- **Intermediate (Tier 2):** 3-4 spells, requires Lore Access 2+.
- **Advanced (Tier 3):** 3-4 spells, requires Lore Access 3+.
- **Master (Tier 4):** 1-3 spells, requires Lore Access 4+. Rare capstone spells — high casting values make stacking multiple Master picks inadvisable, though the rules impose no hard cap on how many a wizard may know.

Casting values are set per spell based on the effect's power, not rigidly by tier. A Basic spell may have a high casting value if the effect is strong; an Intermediate spell may have a low casting value if the effect is modest. Tier gates access (who can learn it); casting value gates reliability (who can cast it consistently).

**Lores come in two types:**
- **General lores** (Heavens, Beasts, Fire, Light, Life, Death, Shadow, Metal): accessible to any qualified wizard. All four tiers present (Lore Access 1-4).
- **Faction lores:** restricted to specific armies and wizard types. Same four-tier structure, but with content unique to the sponsoring faction. A faction-lore caster trades breadth (the general lores) for depth (their faction's signature deeper spells).

> **Note on tier content.** The eight general lores (Heavens, Beasts, Fire, Light, Life, Death, Shadow, Metal) currently carry full Basic / Intermediate / Advanced content. Master-tier content for the general lores is pending a dedicated per-lore magic pass; currently only **Beasts** (The Wild Hunt) and **Life** (The Great Regrowth) carry a single Master spell each as early commits from that pass. The faction lores (Geomancy, Cosmic Architecture) carry full Master content.
