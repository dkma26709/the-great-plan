# Empire — Test Profiles


> **Design principle:** Cross-faction demonstration roster. Drafted to prove the ruleset flexes across different army archetypes (human rank-and-flank vs Lizardmen beast-apex), give a friend's Empire army something to play against, and stress-test mechanics against a faction with gunpowder, Monstrous Cavalry, War Machines, and multi-Wind magic access. See `cross-faction-reference-notes.md` at project root for design-prep research.

> **Heft calibration note for humans.** Empire humans stay close to WFB reference stat lines — they are not "spawned for war" Saurus. WS / S / T / W / I / A values unchanged from reference; **Res gets a +1 bump** across all State Troops and characters (the stress-first system's thresholds mean Res 7 Empire troopers would buckle faster than WFB expects). Shield + armour combinations give 4+ to 3+ combined saves, same one-step-per-source convention as Lizardmen.
